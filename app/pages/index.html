Page Likes Tally

Click the heart to like this page

  • Clicking the heart increments the tally.
  • Clicking the heart again decrements the tally.
  • The tally is stored in a database.
  • The tally is global to all visitors.
  • You can only add one like per session (tracked with a cookie).
  • the session to be able to like more than once.

This demo exercises both database and session interactions and how they can be combined to manage app state

Even the favicon changes depending on your session!

Powered by

This demo is built with Enhance and uses @begin/data to connect to AWS DynamoDB with a dev-friendly API.

Whether running Enhance locally, on Begin.com, or deployed to your own AWS account, the data layer is automatically available.
